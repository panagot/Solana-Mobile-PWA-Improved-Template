Privacy Policy

Reference application - last updated: March 2026

This page describes how the Solana Mobile - Reference PWA demo (the “Application”) handles information. It is provided for transparency and for Solana dApp Store / grant review. Operators who fork this project should replace this document with a policy appropriate to their own product and jurisdiction.

Data we do not operate

This Application does not operate a backend under our control for end-user accounts. We do not sell personal data.

Wallet and blockchain

When you connect a wallet, signing and balance queries are handled by your chosen wallet application and the Solana network (e.g. public RPC endpoints). Public blockchain data may be visible on-chain. Review your wallet provider’s privacy policy for how they process data.

Hosting and analytics

The public demo may be hosted on infrastructure (e.g. Vercel) that collects standard server or platform logs (such as IP address, user agent, and request metadata) under the host’s own policies. This Application does not embed third-party advertising analytics in the reference source tree.

Local storage

The Application may use browser sessionStorage for UI state (e.g. splash or notice dismissal). Progressive Web App features may cache static assets via a service worker.

Contact

For questions about this reference deployment, use the contact channel listed on the application’s GitHub repository or grant submission materials.
